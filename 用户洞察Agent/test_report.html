多标签功能自动化测试

Automated Multi-Tab Functionality Test

如何查看测试结果

  1. 确保已在浏览器中打开 index.html
  2. 按 F12 打开开发者工具
  3. 切换到 Console（控制台） 标签
  4. 测试会自动运行，结果将显示在控制台中
  5. 如需重新运行，在控制台输入：window.runMultiTabTests()
